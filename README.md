# Subaru ECU Definitions

This repository will be housing definitions for Hitachi ECUs. Mainly the M32R based WA12212970WWW MCU that my EUDM Legacy 2.0R uses.  
I am utilizing [FastECU](https://github.com/miikasyvanen/FastECU) for reading and flashing my ECU using an OpenPort 2.0 clone from China.

ECUFlash definitions were omitted in favor of actually open source tooling. It also doesn't support reading and flashing Hitachis.
